Лифтовый симулятор
Программа представляет собой симулятор работы лифтов, написанный на языке программирования Java с использованием Java SDK 15 и библиотеки Swing для создания графического интерфейса.

Описание
Этот лифтовый симулятор моделирует работу лифтов в многоквартирном здании. Он позволяет визуализировать движение лифтов между этажами и их загрузку. Пользователь может наблюдать за работой лифтов и их загрузкой через графический интерфейс программы.

Функциональности
Визуализация лифтов: на графическом интерфейсе отображаются лифты, их текущее положение и состояние.
Загрузка лифтов: пользователь может наблюдать за тем, как люди заходят и выходят из лифтов, отслеживая изменение загрузки лифтов в реальном времени.
Анимация дверей: при открытии и закрытии дверей лифта визуализируется анимация, делающая работу программы более реалистичной.
Информация о загрузке: пользователь может видеть текущую загрузку лифта в формате "текущая загрузка/максимальная загрузка" на графическом интерфейсе.
